# EX-4-Java-program-to-print-the-even-numbers-between-1-20
```
public class EvenNumbers {
    public static void main(String[] args) {
        System.out.println("Even numbers between 1 and 20:");


        for (int i = 1; i <= 20; i++) {

            if (i % 2 == 0) {
                System.out.print(i + " ");
            }
        }
    }
}

```
